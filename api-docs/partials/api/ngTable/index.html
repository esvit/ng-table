ngTable: Table + Angular JS

Installation

First include angular-table.js in your HTML:

<script src="angular.js">
<script src="angular-table.js">

You can download this file from the following places:

  • Google CDN
    e.g. //ajax.googleapis.com/ajax/libs/angularjs/X.Y.Z/angular-table.js
  • Bower
    e.g. 
    bower install angular-table@X.Y.Z
  • code.angularjs.org
    e.g. 
    "//code.angularjs.org/X.Y.Z/angular-table.js"

where X.Y.Z is the AngularJS version you are running.

Then load the module in your application by adding it as a dependent module:

angular.module('app', ['ngTable']);

With that you're ready to get started!
